# 🔐 Secure Login System

A simple secure login system built using Python.  
This project allows users to register and log in with password verification.

---

## 🚀 Features
- User Registration
- User Login
- Password Verification
- Data stored using file handling

---

## 🛠️ Technologies Used
- Python 3
- File Handling

---

## 📂 Project Structure
secure-login-system/
│── main.py
│── users.txt
│── README.md

---

## ▶️ How to Run

1. Install Python 3 on your system
2. Download or clone this repository
3. Open terminal inside the project folder
4. Run:

```bash
python main.py
```
